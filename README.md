# GreenMind AI

GreenMind AI is an AI-powered carbon intelligence platform that helps users track, understand, and reduce their carbon footprint through personalized insights, carbon forecasting, AI recommendations, and sustainability challenges.

## Features

- AI Carbon Twin
- Carbon Footprint Calculator
- Sustainability Dashboard
- Carbon Forecasting
- AI Sustainability Coach
- Gamification System
- Impact Visualization

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Recharts

## Installation

npm install
npm run dev
